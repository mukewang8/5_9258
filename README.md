<h3>课程目标：</h3>
<p>1、有业务决策需求就离不开数据分析，尤其是数据分析思维</p>
<p>从销售、市场，到运营、产品经理、用户研究等，都试图从各种繁杂数据中看出点门道，获得对市场、产品、消费者等方面的洞见。</p>
<p>2、除了数据分析师，更多职位开始对数据分析技能有需求</p>
<p>近50%的岗位需要具备「数据分析」能力，现在几乎大多数热门岗位都会在招聘JD中，给出“具备数据分析能力”这样的招聘条件。</p>
